# card_project
# Homepage (Page 1, “Game Name” )opens with beginning animation and provides a brief overview of the game and links to the game lobby, deck table, and rules. This page serves as the main lobby that players will begin at and return to after a game. 

# Game lobby(Page 2, “Waiting…” ) contains a waiting page to join the game, displaying waiting time, available game lobbies, and online friends. This page also links to the deck table so that the user can check their available decks

# Deck table (Page 3, “Deck Table” )displays available decks to play with, complete with deck stacks and accompanying animations

# Rules (Page 4, “Rules and Tips” ) page contains basic rules for the game, as well as detailed explanations of each card with flip animations for each one. There is also a strategy section with few pointers and tips for playing
